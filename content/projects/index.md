## 🧠 技术特色

- **Multi-Agent 协作：** 每任务 PM→架构→实现→审核→验收，Token 消耗 10-30 倍但质量可靠
- **浏览器自动化：** Puppeteer MCP + Chrome CDP 真实浏览器，不 API 不爬虫
- **大文件处理：** openpyxl 操作 228MB Excel，图片层提取 1000+ 产品图
- **SSH 替代：** 自建 deploy.php HTTP 通道，绕过公司网络 SSH 限制
