# 股票分析模拟交易系统

> **状态：** 🔧 开发中 | **模式：** Multi-Agent 五步流程

## 概述

AI 驱动的短线股票分析与模拟交易系统，采用 Multi-Agent 协作流程进行开发。

## 技术栈

- **前端：** React
- **后端：** Node.js
- **数据库：** PostgreSQL
- **AI 集成：** Claude Code + DeepSeek MiMo

## 开发流程（Multi-Agent）

```
PM Agent → 需求分析
    ↓
架构 Agent → 技术方案设计
    ↓
实现 Agent → 代码编写
    ↓
审核 Agent → Code Review
    ↓
验收 Agent → 功能验收
```

## 核心功能（规划中）

- 实时股票数据接入
- AI 驱动的短线买卖信号
- 模拟交易 / 虚拟资金
- 回测系统
- 收益曲线可视化

## 当前进展

- 架构设计阶段完成
- 技术栈选定
- 后端 API 设计进行中
