# SA 方向机目录

> **状态：** ✅ 已上线 | **迭代版本：** v1 ~ v5 | **数据规模：** 228MB / 1000+ 图片

## 概述

从 228MB 的庞大 Excel 源文件中筛选南美品牌方向机数据，生成格式化产品目录。

## 技术栈

- **语言：** Python
- **Excel 处理：** openpyxl（普通模式 + read_only 模式）
- **图片处理：** PIL（Pillow）
- **数据清洗：** 正则表达式

## 处理流程

```
228MB 源文件 → 品牌筛选（南美） → 图片提取 → 年份清洗 → 格式美化 → 目录输出
```

## 关键数据

| 指标 | 数值 |
|------|------|
| 源文件大小 | 228 MB |
| 品牌数 | 南美目标品牌 |
| 产品图片 | 1000+ |
| 年份覆盖率 | 49/69（正则 v2 改进后） |
| 图片匹配率 | 68/69 |

## 技术挑战

**Excel 图片层模型：** Excel 中的图片是浮动图层，不是单元格内容。openpyxl 的 `read_only` 模式禁用了图片读取。

**解决方案：** 分两步处理：
1. `read_only` 模式读取数据（防止内存溢出）
2. 普通模式加载图片层

## 格式规范

- 列宽：A4/B12/C15/D8/E6/F10/G18/H6/I8/J25/K35
- 品牌色映射：ZF=红、WABCO=黄、Knorr=蓝、Bosch=灰
- 图片高度限制：max_h=45px（与行高一致，防止撑出空行）
- 备注格式：`分类；产地；市场` 三段式
