# 全品类车型数据库

> **状态：** ✅ 已上线 | **数据规模：** 51 品牌 / 1814 行

## 概述

覆盖全球主要汽车品牌的全品类车型数据库，包含中英双语车型名称、品牌、年份、配置等字段。

## 技术栈

- **语言：** Python
- **数据处理：** openpyxl + pandas + 正则表达式
- **数据源：** Wikipedia API
- **翻译：** AI 批量翻译引擎

## 处理流程

```
Wikipedia API → 原始数据 → 清洗/去重 → 中英翻译 → Excel 格式化输出
```

## 数据字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 品牌 | 汽车品牌 | BMW, Toyota, 奔驰 |
| 车型 | 车型名称（中英双语） | 3 Series / 3系 |
| 年份 | 生产年份 | 2015-2020 |
| 类型 | 车身类型 | Sedan, SUV |
| 备注 | 特殊说明 | 国产/进口 |

## 技术亮点

- Wikipedia API 多语言数据拉取
- 正则表达式年份提取（覆盖 1000-2999）
- 品牌色格式化（ZF 红、WABCO 黄、Knorr 蓝等）
- openpyxl 高级样式（列宽、行高、条件格式）
