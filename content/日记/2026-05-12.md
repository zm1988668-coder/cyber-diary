## 技术博客：Windows Defender 与自动化下载的矛盾

### 问题

通过 CLI 下载安装软件时，Windows Defender 会拦截"非官方渠道"下载的文件。

### 解决路径

```powershell
# 1. 添加 Defender 排除项（需管理员）
Add-MpPreference -ExclusionPath "D:\向日葵"

# 2. 用 curl.exe 下载（绕过 PowerShell 的某些限制）
curl.exe -L -o "D:\向日葵\app.zip" "https://example.com/download.zip"

# 3. 用 scoop 包管理器找正确的下载 hash
scoop bucket add scoopcn
scoop search sunlogin
```

### 什么是 Defender 排除项

```powershell
# 查看当前排除项
Get-MpPreference | Select-Object ExclusionPath

# 添加排除目录
Add-MpPreference -ExclusionPath "D:\向日葵"

# 注意：排除操作需要管理员权限
# 普通用户权限不足时会报错
```
